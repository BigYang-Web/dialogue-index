AI Chat TOC Navigator (AI 聊天目录助手)
AI Chat TOC Navigator 是一款专为 AI 聊天用户设计的轻量级浏览器扩展。它能够自动识别并提取对话中的标题层级（H1/H2/H3），在侧边栏生成实时更新的目录，让您在处理长篇 AI 对话（如代码审计、文案撰写、学术研究）时，能像阅读电子书一样轻松导航。
✨ 核心功能

- 🚀 全平台深度适配：完美兼容 ChatGPT, Gemini, DeepSeek, 腾讯元宝, 豆包, 通义千问。
- 📍 智能标题索引：自动解析 AI 回答中的 Markdown 标题，构建清晰的树状导航目录。
- 🔄 实时同步技术：
    - 动态监听：随 AI 逐字生成实时更新目录。
    - 可见性同步：切换标签页后自动刷新数据，确保内容始终最新。
    - SPA 路由适配：在不刷新页面的情况下切换对话列表（如切换会话），目录依然精准匹配。
- ⚡ 精准锚点跳转：点击目录项即刻平滑滚动至目标位置，并提供临时的视觉高亮反馈。
- 🛡️ 隐私与性能：100% 本地运行，不收集任何对话数据；内置防抖逻辑，即使在超长对话下依然保持流畅。

  🛠️ 安装说明
  开发者模式安装 (本地测试)

1. 下载本项目代码压缩包并解压。
2. 打开 Chrome 浏览器，访问 chrome://extensions/。
3. 开启右上角的“开发者模式”。
4. 点击“加载已解压的扩展程序”，选择本项目文件夹。

   🔒 隐私声明
   我们深知 AI 对话内容隐私的重要性：

- 本插件不会上传您的任何聊天记录到外部服务器。
- 所有数据提取、解析和存储工作均在您的浏览器本地完成。
- 权限申请仅限于读取特定 AI 站点的 DOM 结构以生成导航索引。
